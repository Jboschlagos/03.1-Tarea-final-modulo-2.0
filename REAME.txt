Tocar Madera – Demo Ecommerce
=============================

Proyecto frontend desarrollado con:
- HTML
- CSS
- JavaScript
- Bootstrap 5

Funcionalidades:
- Catálogo de productos
- Carrito de compras
- Aplicación de descuento
- Modal funcional
- Compatible con GitHub Pages

Estructura recomendada:

/
│── index.html
│
├── css/
│   └── style.css
│
├── js/
│   └── app.js
│
└── asset/
    └── img/
        ├── mesa.webp
        ├── silla.webp
        └── comedor.webp

Notas:
- Todas las rutas son relativas (./) para compatibilidad con GitHub Pages
- Los nombres de archivos y carpetas deben coincidir exactamente
- No usar rutas absolutas
